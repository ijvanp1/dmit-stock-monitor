## DMIT 是什么，为什么总缺货

DMIT（全称 Depth Master Information Technology）是一家由华人留学生在美国创办的 VPS 商家，主营洛杉矶、香港、东京三个数据中心，核心卖点是三网精品线路：CN2 GIA、9929（联通）、CMIN2（移动）。自建骨干网，直签国内三大运营商——CN2 GIA 带宽 40Gbps，9929 带宽 20Gbps，CMIN2 带宽 20Gbps。这在同价位段的 VPS 商里是拿得出手的硬数据。

硬件方面，全系标配 AMD EPYC 处理器，部分机型已更新到 9005 系列。KVM 虚拟化，不超售。

"不超售"这三个字，就是理解 DMIT 缺货现象的关键。

每一批服务器资源上架，有多少卖多少，卖完下架，等下一批硬件到位才补货。这是良心商家的操作，但对买家来说就变成了手速游戏。热门套餐一上架，几小时内就没了——香港 CN2 GIA 的 VICTORIA 套餐，3 个多小时卖光，有案可查。

另一个缺货原因是 IP 资源。日本 TYO.Pro 系列此前断货，官方客服的解释就是"没有可用 IP 了"。不是机器没了，是 IP 池见底了。等到 IP 资源释放，才会重新放货。

## DMIT 补货有没有规律可循

讲真，DMIT 补货不会提前公告，直接体现在官网库存状态变化上。没有固定周期，但观察历史记录，有几个高频节点：

**节假日是重点。** 黑五、圣诞/元旦、春节、618、双十一——这些节点 DMIT 基本都会配合出限量套餐或者补货一批常规方案。2025 年圣诞促销期间，洛杉矶 T1 系列硬件升级到 AMD EPYC 9005，同时补了 LAX.AN4.Pro.WEE 和 LAX.AN4.EB.CORONA 两款年付 $39.9 起的特价套餐。

**硬件升级周期前后。** DMIT 定期对机器做硬件升级，升级完成后往往会重新放货，而且通常是升级到更新一代的处理器，配置不变价格也不变。

**特殊事件之后。** 2025 年 10 月，香港和东京机房持续遭受 DDoS 攻击，DMIT 处理完问题之后，给受影响老用户赠送了机器，同时开放优惠码让新用户入手。这种情况不常见，但出现了就是一波入手机会。

不同产品线的补货频率差别很大：

- **洛杉矶 T1 系列（LAX.T1）**：相对最稳，通常有库存，是等不了又想试试 DMIT 品质的人最好的入口
- **洛杉矶 Pro/EB 常规套餐**：周期性缺货，但补货频率在精品线路里算高的，一年内有好几次放货记录
- **洛杉矶限量特价套餐（MALIBU、WEE 等）**：缺货最严重，上架即抢，错过等下次
- **香港 Pro（HKG.Pro）**：极度紧俏，补货窗口很短
- **日本 TYO 系列**：Pro 偶有货，TYO.AS3.Pro 在 IP 资源释放后曾补货；Eyeball 系列已于 2026 年 3 月下架

## 不同需求怎么选

**用来搭梯子 / 科学上网：** 电信用户优先 LAX.Pro（CN2 GIA 回程），移动用户优先 LAX.EB（CMIN2 回程），联通两个都行。洛杉矶到国内延迟大概在 150-180ms，晚高峰也保持稳定。

**建站，面向国内用户：** 预算够就香港 Pro（延迟最低），预算有限选洛杉矶 Pro 或 EB，两者线路质量差距不大，价格差挺多的。

**流量需求大但带宽要求不高：** T1 系列。流量用完不停机，自动限速到 50-100Mbps，等月初重置恢复正常，适合跑下载站或流量型业务。

**新手想先试试水：** LAX.T1.WEE 或 HKG.T1.WEE，年付三十多美元，库存相对宽松，用过了你就明白为什么 DMIT 的 Pro 套餐每次补货那么疯抢。
